This is a dynamic and elegant academic personal website template built using HTML, CSS, and JavaScript. It is designed for professors, researchers, or academic professionals who want to showcase their research, publications, teaching, and academic achievements in a modern, responsive format.

#Installation
1. Clone the repository
   --https://github.com/raaakibul/professor-portfolio-site.git
2. Navigate to the project directory
   --cd academic-website
3. Open the index.html file in your web browser or use a local server
   --# Using Python
   --python -m http.server
    Or using Node.js
   --npx http-server
4. Open your browser and visit http://localhost:8000 (or the address provided by your server).
